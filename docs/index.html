What is this?

A minimal prototype showing how cryptographically-verifiable artifacts can backstop ADR workflows. Everything runs in the browser. No server, no cookies. Deterministic outputs, signed payloads, and a transparent Role–Scope model.

  • Static site: host on GitHub Pages or any CDN.
  • Client-side signing & verification; exportable proofs.
  • Composable presets (see fallback-preset.js).

Quickstart

  1. Open this page on GitHub Pages.
  2. Go to Role–Scope Matrix to generate an escrow profile.
  3. Export the signed JSON; share it with counterparties.
